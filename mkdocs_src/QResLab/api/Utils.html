    Utils.FlagPoleState(dim, p)¶

    FlagPoleState Provide a flag pole state

    This function produces a flag pole state.

    \[|\psi\rangle = \sqrt{p}|0\rangle + \sum_{i=1}^{d-1} \sqrt{\frac{1-p}{d-1}} |i\rangle\]
    Parameters:¶

    • dim (numeric) – Dimension of the quantum state.

    • p (numeric) – The strength of the flag.

    Returns:¶

    The density matrix of the flag pole state.

    Return type:¶

    state (matrix)

    Reference:¶

    Díaz, M. G., Fang, K., Wang, X., Rosati, M., Skotiniotis, M., Calsamiglia, J., & Winter, A. (2018). Using and reusing coherence to realize quantum processes. Quantum, 2, 100.

    Requires:¶

    None.

    Author:¶

    Benchi Zhao

    Last updated:¶

    April 25, 2024
