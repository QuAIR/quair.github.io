    error_mitigation¶

    error_mitigation.ProbErrorCancel(Noise_channel)¶

    Produce an HPTP map (Choi) to inverse the Noise_channel.

    ..math:

    \mathcal{N}^{-1} = c_1 \mathcal{D}_1 + c_2 \mathcal{D}_2.
    Parameters:

    Noise_channel (matrix) – The Choi matrix of the noise channel.

    Returns:

    The sampling overhead to realize the inverse channel \(\mathcal{N^-1}\), which is \(c_1+c_2\). JD (matrix): The choi matrix of the inverse channel \(\mathcal{N^-1}\). J1 (matrix): The choi matrix of the channel \(\mathcal{D}_1\). J2 (matrix): The choi matrix of the channel \(\mathcal{D}_2\).

    Return type:

    overhead (numeric)

    Note

    Temme, Kristan, Sergey Bravyi, and Jay M. Gambetta. Error mitigation for short-depth quantum circuits. Physical review letters 119.18 (2017): 180509.

    error_mitigation.ProbErrorCancelO(Noise_channel, O)¶

    Produce an HPTP map (Choi) to inverse the Noise_channel with respect to observable O.

    ..math:

    \mathcal{N}^\dagger\circ\mathcal{D}^\dagger(O) = O,
    Parameters:

    • Noise_channel (matrix) – The Choi matrix of the noise channel.

    • O (matrix) – Observable.

    Returns:

    The sampling overhead to realize the HPTP map \(\mathcal{D}\), which is \(c_1+c_2\). JD (matrix): The choi matrix of the inverse channel \(\mathcal{D}\). J1 (matrix): The choi matrix of the channel \(\mathcal{D}_1\). J2 (matrix): The choi matrix of the channel \(\mathcal{D}_2\).

    Return type:

    overhead (numeric)

    Note

    Zhao, X., Zhao, B., Xia, Z., & Wang, X. (2023). Information recoverability of noisy quantum states. Quantum, 7, 978.

    Provide a CPTP map to inverse the Noise_channel repect to observable O with the method called observable shift.

    ..math:

    \mathcal{N}^\dagger\circ\mathcal{D}^\dagger(O) = \frac{1}{f} (O +
tI)
    Parameters:

    • Noise_channel (matrix) – The Choi matrix of the noise channel.

    • O (matrix) – Observable.

    Returns:

    The sampling overhead to retrive classical information, which equals to \(f\). t (numeric): The shifted distance. JD (matrix): The choi matrix of the target channel to implement.

    Return type:

    overhead (numeric)

    Note

    Zhao, B., Jing, M., Zhang, L., Zhao, X., Wang, K., & Wang, X. (2023). Retrieving non-linear features from noisy quantum states. arXiv preprint arXiv:2309.11403.
